Website: The Kathmandu Post
URL: https://kathmandupost.com/
Date of Scraping: 2023-07-24 

Scraped Data:
- The scraped data is saved in the "kathmandupost_province5.csv" and "kathmandupost_province2.csv" file.
- The CSV file contains the following columns:
  - Article Title
  - Article Content
  - Date
  - Location
  - Author

Scraping Process:
- The scraping was done using Python with the following libraries:
  - Selenium (for dynamic content and "Load More" button)
  - BeautifulSoup (for parsing the HTML content)
  - pandas (for creating and saving the CSV file)

Note:
- The data is scraped for educational and demonstration purposes only.
- Make sure to review and adhere to the website's terms of service and scraping guidelines before using the code for any other purposes.
